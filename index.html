HTML by Alphabet

"Welcome to HTML by Alphabet!"

  • < !--...--> - Defines a comment.
  • < !DOCTYPE> - Defines the document type.
  • < a> - Defines a hyperlink.

HTML by Category

"Welcome to HTML by Category!"

  • < html> - Defines an HTML document.
  • < head> - Contains metadata/information for the document.
  • < title> - Defines a title for the document.

HTML Attributes

"Welcome to HTML Attributes!"

  • accept (attribute) belongs to < input> - Specifies the types of files that the server accepts (only for type="file" ).
  • accept-charset (attribute) belongs to < form> - Specifies the character encodings that are to be used for the form submission.
  • accesskey (attribute) belongs to Global Attributes - Specifies a shortcut key to activate/focus an element.

HTML Events

"Welcome to HTML Events!"

  • onafterprint (attribute) script (value) - Script to be run after the document is printed.
  • onbeforeprint (attribute) script (value) - Script to be run before the document is printed.
  • onbeforeunload (attribute) script (value) - Script to be run when the document is about to be unloaded.

HTML Canvas

"Welcome to HTML Canvas!"

  • fillStyle - Sets or returns the color, gradient, or pattern used to fill the drawing.
  • strokeStyle - Sets or returns the color, gradient, or pattern used for strokes.
  • shadowColor - Sets or returns the color to use for shadows.
